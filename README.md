
# 📱 **BharatPe Merchant Acquisition & Retention Analysis**

## 🎯 **Project Overview**

This project focuses on **Merchant Acquisition & Retention Analysis** for BharatPe, one of India's leading fintech companies. The analysis aims to identify key factors affecting merchant acquisition and retention rates, providing actionable insights to improve business performance.

---

## 🚀 **Project Objective**

Create a comprehensive analysis of factors affecting merchant acquisition and retention for BharatPe, with data-driven recommendations to enhance both metrics and support strategic decision-making.

---

## 📊 **Data Sources**

- **🎯 Play Store Reviews**: Scraped customer reviews and ratings data
- **🌐 Website Data**: Collected merchant and transaction-related information  
- **⚡ Synthesized Data**: Generated additional datasets for comprehensive analysis

---

## 🔍 **Project Methodology**

### **Step 1: Data Collection Strategy**
- Web scraping of Play Store reviews for customer sentiment analysis
- Collection of website data for merchant behavior patterns
- Data synthesis to fill gaps and enhance analysis depth

### **Step 2: Data Processing & Analysis**
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Statistical analysis of acquisition and retention patterns
- Customer sentiment analysis from reviews

### **Step 3: Visualization & Dashboard Creation**
- Interactive dashboards for key metrics
- Trend analysis visualizations
- Merchant segmentation charts
- Performance comparison graphics

### **Step 4: Business Insights & Recommendations**
- Identification of key retention factors
- Analysis of acquisition channel effectiveness
- Customer satisfaction correlation with retention
- Market positioning insights

### **Step 5: Presentation & Documentation**
- Executive summary of findings
- Detailed technical documentation
- Business recommendations report

## 🚀 **Getting Started**

### **📋 Prerequisites**
```bash
Python 3.7+
pip package manager
```

### **⚙️ Installation**

**1. Clone the repository**
```bash
git clone https://github.com/yourusername/bharatpe-analysis.git
cd bharatpe-analysis
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Run the analysis**
```bash
python src/main.py
```

```bash
streamlit run dashboard/app.py
```
